## Mods included

### VehicleLimitMod
Raises the max vehicle limit to 2000. Research **"Max Vehicle Limit"** in the science tree (after VehicleCapIncrease6) to unlock it.

### SandboxUnlockMod
Unlocks Product Source and Item Sink buildings outside of sandbox mode. Research **"Logistics Infrastructure"** (after Maintenance Depot) to unlock all 8 port-type variants.

### OverclockMod WIP
Set any building's production speed from 0% to 1000%.

**Usage:**
1. Open the console with the backtick key
2. Click the building you want to overclock
3. Type `overclock 500` to set it to 500%
4. Type `overclock_status` to check the current value

### WoodNuclearMod
Adds a new building that burns Wood, Water, and Diesel to produce SteamHi and Slag.

## Installation

1. Copy each mod folder into `%APPDATA%\Captain of Industry\Mods\`
2. Launch the game
3. Go to **Settings → Miscellaneous → Enable Mods**
4. Restart and select your mods when starting or loading a game

## Requirements

- Captain of Industry v0.8.3+
- Trains DLC (for some mods)
